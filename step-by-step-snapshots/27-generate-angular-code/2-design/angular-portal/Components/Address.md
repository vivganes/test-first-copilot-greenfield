# Address

## Properties
- `type: string` — Address type (Shipping, Billing).
- `line1: string` — First line of address.
- `line2: string` — Second line of address (optional).
- `city: string` — City name.
- `state: string` — State or region.
- `postalCode: string` — Postal or ZIP code.
- `country: string` — Country name.
- `isDefault: boolean` — Indicates if this is the default address.
- `phoneNumber: string` — Phone number for delivery (optional).

- None (data model)

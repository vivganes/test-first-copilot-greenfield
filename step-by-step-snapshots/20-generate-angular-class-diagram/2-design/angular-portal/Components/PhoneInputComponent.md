# PhoneInputComponent

## Properties
- `phoneForm: FormGroup` — Form group for phone number input.
- `phoneNumber: PhoneNumber` — Model representing the phone number.

## Methods
- `buildPhoneForm()` — Initializes phone form group and validators.
- `getPhoneValue(): PhoneNumber` — Returns the current phone number value.
- `validatePhone()` — Validates phone number field.

## Related Classes
- `RegistrationFormComponent`: Uses this component for phone input.
- `PhoneNumber`: Data model for phone number details.

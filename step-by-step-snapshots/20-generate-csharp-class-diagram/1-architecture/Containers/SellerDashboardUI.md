# Seller Dashboard UI (Micro-Frontend)

## Description
The Seller Dashboard UI is a web application for sellers to manage their products, view sales, handle orders, and respond to support tickets. It provides analytics and inventory management features, communicating with backend services via the API Gateway.

## Communication
- Communicates with: API Gateway, Product Service, Order Service, Inventory Service, Promotion Service, Notification Service, Analytics & Audit Service, Support Service.

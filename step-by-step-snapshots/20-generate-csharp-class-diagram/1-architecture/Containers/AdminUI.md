# Admin UI (Micro-Frontend)

## Description
The Admin UI is a web application for platform administrators. It provides interfaces for managing users, products, orders, promotions, analytics, and support tickets. Admins can monitor system health, review audit logs, and configure platform settings. It communicates with backend services via the API Gateway.

## Communication
- Communicates with: API Gateway, User Service, Product Service, Order Service, Inventory Service, Promotion Service, Notification Service, Analytics & Audit Service, Support Service.

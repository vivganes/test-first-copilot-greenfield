# Compliance & Regionalization Service

## Description
The Compliance & Regionalization Service manages integration with external compliance APIs (tax, privacy, regional regulations) and handles regional logic such as tax calculation, multi-currency support, and privacy compliance (GDPR/CCPA). It ensures the platform operates legally and correctly in all supported regions.

## Communication
- Communicates with: Order Service, Payment Service, User Service, API Gateway, external Compliance APIs (tax, privacy), Admin UI.

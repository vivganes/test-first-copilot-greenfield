# useNotification

**Description:**
Custom hook for showing toast/snackbar notifications.

## Properties (State)
- `message`: Current notification message
- `type`: Notification type (success, error, info)
- `visible`: Boolean

## Methods
- `show(message, type)`: Shows notification
- `hide()`: Hides notification

## Related Classes
- **RegistrationForm**: Uses for feedback
- **SuccessMessage**: May use for success feedback

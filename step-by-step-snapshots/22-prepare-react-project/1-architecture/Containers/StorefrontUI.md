# Storefront UI (Micro-Frontend)

## Description
The Storefront UI is the customer-facing web application. It allows users to browse products, manage their cart, place orders, write reviews, and manage their account. It communicates with backend services via the API Gateway.

## Communication
- Communicates with: API Gateway, User Service, Product Service, Shopping Cart Service, Order Service, Review & Rating Service, Promotion Service, Notification Service, Payment Service, Shipping Service, Support Service.

# UserRegistrationResponse

## Properties
- `userId: string` — Unique identifier for the registered user.
- `message: string` — Success message from API.
- `emailVerified: boolean` — Whether the user's email is verified.
- `status: string` — Account status (e.g., Active).
- `role: string` — User role (e.g., Customer).

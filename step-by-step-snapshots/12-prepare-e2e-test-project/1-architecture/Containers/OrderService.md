# Order Service

## Description
The Order Service handles the entire order lifecycle, from order placement to delivery and returns. It manages order status, payment association, shipment tracking, and order history. It ensures business rules for order processing, refunds, and cancellations are enforced.

## Communication
- Communicates with: Payment Service, Shipping Service, Inventory Service, User Service, Notification Service, API Gateway, Storefront UI, Seller Dashboard UI, Admin UI.

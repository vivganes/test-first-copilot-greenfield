# PhoneNumberField

**Description:**
Input component for phone number with validation and formatting.

## Properties
- `value`: Phone number value
- `onChange`: Change handler
- `error`: Error message
- `required`: Boolean

## Methods
- `formatPhoneNumber(value)`: Formats phone number for display
- `validatePhoneNumber(value)`: Validates phone number

## Related Classes
- **RegistrationForm**: Uses this for phone input
- **validationUtils**: For phone validation
- **FormErrorMessage**: For error display

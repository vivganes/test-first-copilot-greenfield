# RegistrationService

**Description:**
Service for handling registration API calls.

## Properties
- None (stateless service)

## Methods
- `registerUser(data)`: Calls backend API to register user
- `checkEmailUnique(email)`: Checks if email is unique

## Related Classes
- **apiClient**: Used for HTTP requests
- **useApi**: Used for async calls
- **RegistrationForm**: Uses for registration

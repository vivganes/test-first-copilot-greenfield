# User Service

## Description
The User Service manages all user-related operations, including registration, authentication, profile management, role-based access, and account deletion. It handles Customers, Sellers, and Admins, ensuring secure storage of user credentials and compliance with privacy regulations (e.g., GDPR/CCPA). It also manages user addresses and supports social login integrations.

## Communication
- Communicates with: Authentication Service, Notification Service, API Gateway, Order Service, Shopping Cart Service, Review & Rating Service, Support Service, Admin UI, Storefront UI, Seller Dashboard UI.

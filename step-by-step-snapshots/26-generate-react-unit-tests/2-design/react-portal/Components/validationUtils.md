# validationUtils

**Description:**
Utility functions for validating registration fields.

## Properties
- None (pure functions)

## Methods
- `validateEmail(email)`: Checks format
- `validatePassword(password)`: Checks rules
- `validateAddress(address)`: Checks address fields
- `validatePhone(phone)`: Checks phone format

## Related Classes
- **RegistrationForm**: Uses for validation
- **RegistrationField**: Uses for field validation
